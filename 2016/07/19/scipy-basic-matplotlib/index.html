Matplotlib：plotting

简介

matplotlib作为在Python下最简单的二维图像绘制包，它给我们提供了方便的接口以快速实现数据的可视化和对多种格式的图像进行操作。

设置Ipython的matplotlib模式

为了使matplotlib展示图片时弹出对话框打断我们的编码，可以设置matplotlib在python中以插入模式运行，开启方法如下：

  • 使用Ipython --matplotlib启动IPython
  • 在IPython NoteBook(Jupyter)中，输入%matplotlib inline

pyplot

pyplot给Python提供了一个面向对象的可编程绘图库，它模仿了Matlab的一些操作，所以熟悉Matlab的同学可以很快的上手。pyplot的导入方式如下所示

from matplotlib import pyplot as plt

Figure, Subplots, Axes 和 Ticks

Figure

与Python数组不同，Figure的编号从1开始，这更接近于MATLAB的风格。
参数：

除了点击GUI的X来关闭显示窗口之外，还可以通过调用plt.close(num)方法来关闭，如果没有参数则关闭当前窗口，带有数字参数则关闭指定窗口，如果参数是“all”则关闭所有窗口。

子图 subplot

相比较使用subplot，更强大的是gridspec类

  • 使用plt.subplot2grid()

    ax1 = plt.subplot2grid((3,3), (0,0), colspan=3)
ax2 = plt.subplot2grid((3,3), (1,0), colspan=2)
ax3 = plt.subplot2grid((3,3), (1, 2), rowspan=2)
ax4 = plt.subplot2grid((3,3), (2, 0))
ax5 = plt.subplot2grid((3,3), (2, 1))

  • 使用gridspec.GridSpec

    import matplotlib.gridspec as gridspec
gs = gridspec.GridSpec(3, 3)
ax1 = plt.subplot(gs[0, :])
ax2 = plt.subplot(gs[1,:-1])
ax3 = plt.subplot(gs[1:, -1])
ax4 = plt.subplot(gs[-1,0])
ax5 = plt.subplot(gs[-1,-2])

    图同上

  • 调整GridSpec的布局

    gs1 = gridspec.GridSpec(3, 3)
gs1.update(left=0.05, right=0.48, wspace=0.05)
ax1 = plt.subplot(gs1[:-1, :])
ax2 = plt.subplot(gs1[-1, :-1])
ax3 = plt.subplot(gs1[-1, -1])

gs2 = gridspec.GridSpec(3, 3)
gs2.update(left=0.55, right=0.98, hspace=0.05)
ax4 = plt.subplot(gs2[:, :-1])
ax5 = plt.subplot(gs2[:-1, -1])
ax6 = plt.subplot(gs2[-1, -1])

    调整gs1和gs2的布局，使他们各占一半空间

    import numpy as np
from matplotlib import pyplot as plt
def f(x, y):
    return (1 - x / 2 + x ** 5 + y **3) * np.exp(-x ** 2 - y ** 2)
n = 256
x = np.linspace(-3, 3, n)
y = np.linspace(-3, 3, n)
X, Y = np.meshgrid(x, y)
'''

在轮廓内部填充颜色

'''
plt.contourf(X, Y, f(X, Y), 8, alpha=0.75, cmap='jet')
'''

画出轮廓线

'''
C = plt.contour(X, Y, f(X, Y), 8, colors='black', linewith=0.5)
plt.clabel(C, inline=1, fontsize=19)
