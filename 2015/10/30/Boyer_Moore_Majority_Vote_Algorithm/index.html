еҒҮеҰӮжңүдёҖдёӘй•ҝеәҰдёәNзҡ„д№ұеәҸж•°з»„пјҢеёҢжңӣеҲӨж–ӯе…¶дёӯжҳҜеҗҰеӯҳеңЁвҖқдё»е…ғзҙ вҖқ(еҮәзҺ°ж¬Ўж•°еӨ§дәҺ$\lfloor{\frac{N}{2}}\rfloor$зҡ„е…ғзҙ )

жң¬ж–ҮеҸӮиҖғзҝ»иҜ‘иҮӘMajority Voting Algorithm Find the majority element in a list of values

й—®йўҳзҡ„ж„Ҹд№ү

еңЁйҖҡдҝЎдёӯпјҢдёәдәҶйҳІжӯўдҝЎйҒ“еҷӘеЈ°е№Іжү°еҜјиҮҙдј иҫ“еҮәй”ҷпјҢеёёеёёдјҡиҝһз»ӯеҸ‘йҖҒеӨҡж¬ЎеҗҢж ·зҡ„дҝЎеҸ·з»ҷжҺҘж”¶з«ҜпјҢжҺҘж”¶з«Ҝж”¶еҲ°иҝҷдәӣдҝЎеҸ·еҗҺпјҢе°ҶйҮҚеӨҚеҮәзҺ°ж¬Ўж•°жңҖеӨҡзҡ„жҺҘж”¶дҝЎеҸ·еҪ“еҒҡжӯЈзЎ®дҝЎжәҗдҝЎеҸ·гҖӮ

з®ҖеҚ•зҡ„и§Јжі•

жҲ‘д»¬еҸҜд»Ҙе…Ҳе°Ҷд№ұеәҸж•°з»„жҺ’еәҸпјҢеҰӮжһңж•°з»„дёӯеӯҳеңЁдё»е…ғзҙ пјҢеҲҷзҺ°еңЁдҪҚдәҺдёӯй—ҙзҡ„е…ғзҙ дёҖе®ҡжҳҜдё»е…ғзҙ гҖӮиҝҷдёӘз®—жі•з”ұдәҺз”ЁеҲ°дәҶжҺ’еәҸпјҢжүҖд»Ҙж—¶й—ҙеӨҚжқӮеәҰиҮіе°‘жҳҜ$O(NlgN)$гҖӮ

дҪҶжҳҜжҲ‘д»¬жңүжӣҙеҘҪзҡ„еҠһжі•пјҢеҸҜд»ҘиҫҫеҲ°зәҝжҖ§ж—¶й—ҙеӨҚжқӮеәҰ$O(N)$з©әй—ҙеӨҚжқӮеәҰ$O(1)$гҖӮ

Boyer-Moore жңҖеӨ§жҠ•зҘЁз®—жі•

жңҖеӨ§жҠ•зҘЁз®—жі•йңҖиҰҒйҒҚеҺҶдёӨж¬Ўж•°з»„пјҢз¬¬дёҖж¬ЎжүҫеҲ°ж•°з»„дёӯеҮәзҺ°ж¬Ўж•°жңҖеӨҡзҡ„е…ғзҙ пјҢз¬¬дәҢж¬Ўи®Ўз®—иҜҘе…ғзҙ еҮәзҺ°зҡ„ж¬Ўж•°пјҢеҰӮжһңеҮәзҺ°ж¬Ўж•°еӨ§дәҺ$\lfloor{\frac{N}{2}}\rfloor$еҲҷиҜҘе…ғзҙ жҳҜдё»е…ғзҙ пјҢеҗҰеҲҷдё»е…ғзҙ дёҚеӯҳеңЁ[^ 1]гҖӮ

еңЁз¬¬дёҖж¬ЎйҒҚеҺҶдёӯпјҢйңҖиҰҒдёӨдёӘеҸҳйҮҸпјҢз¬¬дёҖдёӘеҸҳйҮҸcandidateз”ЁжқҘдҝқеӯҳеҖҷйҖүе…ғзҙ пјҢз¬¬дәҢдёӘеҸҳйҮҸcountз”ЁдәҺи®Ўж•°пјҢе®ғиў«еҲқе§ӢеҢ–дёә0гҖӮ

йҒҚеҺҶж•°з»„дёӯзҡ„жҜҸдёӘе…ғзҙ пјҢйҰ–е…ҲжЈҖжҹҘcountжҳҜеҗҰжҳҜ0пјҢеҰӮжһңжҳҜзҡ„иҜқе°Ҷcandidateи®ҫдёәеҪ“еүҚйҒҚеҺҶзҡ„е…ғзҙ пјӣз„¶еҗҺеҲӨж–ӯеҪ“еүҚйҒҚеҺҶе…ғзҙ жҳҜеҗҰзӯүдәҺcandidate,зӯүдәҺзҡ„иҜқcountеҠ 1пјҢеҗҰеҲҷеҮҸ1гҖӮиҝҷдёҖжӯҘpython д»Јз ҒеҰӮдёӢпјҡ

дёҠйқўзҡ„еҫӘзҺҜз»“жқҹеҗҺпјҢеҪ“еүҚcandidateе°ұжҳҜдё»е…ғзҙ пјҲеҰӮжһңж•°з»„дёӯеӯҳеңЁдё»е…ғзҙ зҡ„иҜқпјүгҖӮдёӢйқўжҲ‘д»¬и§ЈйҮҠдёҖдёӢе®ғзҡ„еҺҹзҗҶгҖӮ

еҺҹзҗҶ

жҲ‘д»¬еҸӘйңҖиҰҒиҖғиҷ‘дё»е…ғзҙ еӯҳеңЁзҡ„жғ…еҶөпјҢеӣ дёәдё»е…ғзҙ дёҚеӯҳеңЁзҡ„жғ…еҶөдёӢпјҢз¬¬дәҢйҒҚеҫӘзҺҜдјҡе°Ҷе…¶жҺ’йҷӨгҖӮ

йҰ–е…ҲжҲ‘д»¬иҖғиҷ‘з¬¬дёҖдёӘе…ғзҙ дёҚжҳҜдё»е…ғзҙ зҡ„жғ…еҶө,жҜ”еҰӮпјҡ

еҪ“йҒҚеҺҶз¬¬дёҖдёӘе…ғзҙ ж—¶пјҢcandidateиў«и®ҫдёә5пјҢcountиў«и®ҫдёә1пјҢд№ӢеҗҺиӢҘйҒҮеҲ°5countеҠ 1,еҗҰеҲҷеҮҸ1(зӣёеҪ“дәҺйқһcandidateе…ғзҙ вҖқж¶ҲиҖ—вҖқжҺүдәҶcandidateе…ғзҙ  ),зӣҙеҲ°countеҸҳжҲҗ0пјҢиҝҷж—¶пјҢйқһcandidateе…ғзҙ вҖқж¶ҲиҖ—вҖқжҺүзӯүйҮҸзҡ„candidateе…ғзҙ пјҢжҜ”еҰӮдёҠйқўзҡ„дҫӢеӯҗпјҢеҪ“йҒҚеҺҶеҲ°з¬¬4дёӘе…ғзҙ ж—¶countеҸҳдёә0пјҡ

иҝҷдёӘдҫӢеӯҗдёӯжҲ‘д»¬дҪҝз”ЁдәҶдё»е…ғзҙ вҖқж¶ҲиҖ—вҖқcandidate,и§ӮеҜҹж•°з»„еү©дҪҷйғЁеҲҶ[0, 5, 0, 0, 5],е®Ңж•ҙж•°з»„зҡ„дё»е…ғзҙ 0д»Қз„¶жҳҜеү©дҪҷйғЁеҲҶзҡ„дё»е…ғзҙ пјӣеҰӮжһңжҲ‘д»¬з”ЁдәҺж¶ҲиҖ—candidateзҡ„е…ғзҙ дёҚжҳҜдё»е…ғзҙ пјҢиҝҷдёӘз»“и®әжҳҫз„¶жӣҙеҠ жҲҗз«ӢгҖӮ

еҗҢж ·зҡ„пјҢеҰӮжһңз¬¬дёҖдёӘе…ғзҙ жҳҜдё»е…ғзҙ пјҢеҪ“countеҸҳдёә0ж—¶пјҢе®Ңж•ҙж•°з»„зҡ„дё»е…ғзҙ д»Қз„¶жҳҜеү©дҪҷйғЁеҲҶзҡ„дё»е…ғзҙ гҖӮ

иҝҷж ·пјҢйҡҸзқҖйҒҚеҺҶзҡ„иҝӣиЎҢпјҢжҲ‘д»¬дёҚж–ӯйҮҚеӨҚдёҠйқўзҡ„иҝҮзЁӢпјҢзӣёеҪ“дәҺжҠҠеҫ…жұӮж•°з»„дёҚж–ӯзј©зҹӯпјҢзӣҙеҲ°countе…¶д»–е…ғзҙ дёҚиғҪе°ҶcandidateвҖңж¶ҲиҖ—вҖқдёә0гҖӮ

еҰӮжһңдёҖдёӘж•°з»„дёҚиғҪе°ҶcountеҸҳдёә0пјҢеҲҷзӯүдәҺз¬¬дёҖдёӘе…ғзҙ зҡ„е…ғзҙ ж•°зӣ®дёҖе®ҡеӨҡдәҺдёҚзӯүдәҺз¬¬дёҖдёӘе…ғзҙ зҡ„е…ғзҙ ж•°зӣ®пјҢиҖҢз¬¬дёҖдёӘе…ғзҙ зҡ„еҖјпјҲcandidateзҡ„еҖјпјүе°ұжҳҜеҪ“еүҚдё»е…ғзҙ гҖӮ

жӣҙе°‘жҜ”иҫғзҡ„ж–№жі•
