编译命令

clang++ -std=c++11 -stdlib=libc++ -Weverything main.cpp

要使用gnu特性，将”-std=c++11”改为”-std=gnu++11”

Main函数

  int main() {
    return 0;
  }

0表示正确执行，-1表示错误

输出类型

  • cin
  • cout
  • cerror
  • clog

连续读入数值

  #include
  int main() {
    int val, rst;
    while(std::cin>>val){
      rst+=val;
    }
    return 0;

头文件

对于标准库的头文件用”<>”包围且不需要”.h”，而自定义的头文件需要用""包围，且需要.h后缀

重定向

为了减少无谓的重复输入测试数据，可以使用重定向
$addItems < infile > outfile
