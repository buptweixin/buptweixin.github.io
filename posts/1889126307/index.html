CS231n assignment1 multi svm -- 矩阵乘法的几何意义


问题引入

在cs231n的assignment1多类svm部分中，需要采用完全向量法求解损失函数对权重的梯度$dW$,$$\nabla{w{y_i}}Li=-\left(\sum{j\neq{y_i}}1(w_j^Txi-w{y_i}^Tx_i+\Delta>0)\right)xi\\nabla{w_j}L_i = 1\left(w_j^Txi-w{y_i}^T+\Delta\right)x_i$$(详细推导),即给出输入$x_i$，假设正确类别是$y_i$,则权重的$y_i$列的梯度为满足$w_j^Txi-w{y_i}^Tx_i+\Delta>0$的列数量乘输入’$-x_i$’，其他列的梯度为当$w_j^Txi-w{y_i}^Tx_i+\Delta>0$等于$x_i$,否则为0.如何用矩阵方法解决得到答案呢？

矩阵相乘的意义

假设有$2\times{3}$矩阵A和$3\times{2}$矩阵B相乘，我们知道结果如下：

$$\left[\begin{array}{ccc}x11 & x12 & x13 \x21 & x22 & x23\end{array}\right] \times\left[\begin{array}{cc}y11 & y12 \y21 & y22 \y31 & y32\end{array}\right]=\left[\begin{array}{cc}x{11}y{11}+x{12}y{21}+x{13}y{31} & x{11}y{12}+x{12}y{22}+x{13}y{32} \x{21}y{11}+x{22}y{21}+x{23}y{31} & x{21}y{12}+x{22}y{22}+x{23}y{32}\end{array}\right]$$

那么它的意义是什么呢？

我们仔细观察等式右边的每列，会发现同一列的B矩阵元素$y{ij}$都是相等的

$$\left[\begin{array}{cc}\left[\begin{array}{c}x{11} \x{21}\end{array}\right]y{11}+\left[\begin{array}{c}x{12} \x{22}\end{array}\right]y{21}+\left[\begin{array}{c}x{13} \x{23}\end{array}\right]y{31} & \left[\begin{array}{c}x{11} \x{21}\end{array}\right]y{12}+\left[\begin{array}{c}x{12} \x{22}\end{array}\right]y{22}+\left[\begin{array}{c}x{13} \x{23}\end{array}\right]y_{32}\end{array}\right]$$

这就是说，对于矩阵$A\times{B}$,结果的第j列相当于矩阵$A$的每一列分别乘以矩阵$B$第j列第一到最后一个元素然后相加。

所以？

我们将$-x_i$乘以margin大于0的数量用来累加$x_i$正确类别对应的权重梯度，其他列根据margin是否大于0来累加，若大于0，累加对应列的梯度，否则不累加。

具体步骤:

  1. 使用包含n个D维数据的矩阵乘D行c列(c为分类类别数)权重W得到每个样本分成每一个类别的分数(第i行第j个元素代表第i个样本被分类到第j类的分数)；
  2. 将每个数据的所有类别的分数都减去正确类别的分数再加上$\Delta$,并把正确类别对应项置为0得到margin；
  3. 判断margin是否大于0，计算每一个输入数据得到的大于0的margin的数量
  4. 将输入转置，每列代表一个数据，我们需要通过右边的矩阵指定每个数据累加的次数。

博文给出的代码如下：

def svm_loss_vectorized(W, X, y, reg):
  """
  Structured SVM loss function, vectorized implementation.

  Inputs and outputs are the same as svm_loss_naive.
  """
  loss = 0.0
  dW = np.zeros(W.shape) # initialize the gradient as zero

  #print(type(W))
  #print(type(X))

  num_classes = W.shape[1]
  num_train = X.shape[0]

  scores = X.dot(W)
  correct_class_score = scores[np.arange(num_train), y]

  #print(scores.shape)
  #print(correct_class_score.shape)
  #print(type(scores))
  #print(type(correct_class_score))

  tmpMat = scores.T - correct_class_score + 1
  tmpMat = tmpMat.T
  tmpMat[np.arange(num_train), y] = 0

  margin = np.maximum(tmpMat, np.zeros((num_train, num_classes)))

  #print(margin.shape)
  loss = np.sum(margin)
  loss /= num_train
  loss += 0.5*reg*np.sum(W*W)

  # Binarize into integers
  binary = margin
  binary[margin > 0] = 1

  # Perform the two operations simultaneously
  # (1) for all j: dW[j,:] = sum_{i, j produces positive margin with i} X[:,i].T
  # (2) for all i: dW[y[i],:] = sum_{j != y_i, j produces positive margin with i} -X[:,i].T
  col_sum = np.sum(binary, axis=1)
  binary[np.arange(num_train), y] = -col_sum[range(num_train)]
  dW = np.dot(X.T, binary)

  # Divide
  dW /= num_train

  # Regularize
  dW += reg*W

  return loss, dW
