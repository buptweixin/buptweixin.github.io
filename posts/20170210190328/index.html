32.Longest Valid Parentheses

Problem Description

Given a string containing just the characters '(' and ')', find the length of the longest valid (well-formed) parentheses substring.

For "(()", the longest valid parentheses substring is "()", which has length = 2.

Another example is ")()())", where the longest valid parentheses substring is "()()", which has length = 4.

给定一个只包含(和)的字符串，找出符合括号配对规则的最大合法长度

Method

遍历字符串，如果遇到(将其所在位置压栈，遇到)的话，首先判断栈是否为空，是空的话)非法，将其序号压入栈中，否则需要继续判断栈顶是否是(，如果是的话表示配对合法，直接将(序号出栈，否则该)没有配对成功，将其入栈。遍历完整个字符串之后，如果栈为空，说明字符串中所有括号都配对成功，直接返回字符串长度即可，否则，计算没对非法括号之间的距离，其中最大的即为答案。

336. Palindrome Pairs

Problem Description

Given a list of unique words, find all pairs of distinct indices (i, j) in the given list, so that the concatenation of the two words, i.e. words[i] + words[j] is a palindrome.

Example 1:

Given words = ["bat", "tab", "cat"]

Return [[0, 1], [1, 0]]

The palindromes are ["battab", "tabbat"]

Example 2:

Given words = ["abcd", "dcba", "lls", "s", "sssll"]

Return [[0, 1], [1, 0], [3, 2], [2, 4]]

The palindromes are ["dcbaabcd", "abcddcba", "slls", "llssssll"]

给出一组字单词数组，找出所有能构成回文的单词对。

Method

其原理为：

如果两个单词能拼接成回文的话，那么较短的单词一定会以逆序的形式出现在较长的单词的开头或者末尾，即palindrome = shorter+ longer = shorter + (middle + reverse(shorter)) orpalindrome = longer + shorter = (reverse(shorter) + middle) + shorter

214. Shortest Palindrome

Problem Description

Given a string S, you are allowed to convert it to a palindrome by adding characters in front of it. Find and return the shortest palindrome you can find by performing this transformation.

For example:

Given "aacecaaa", return "aaacecaaa".

Given "abcd", return "dcbabcd".

给出一个字符串，在其左边加入最少的字符使其构成回文

Method

直接判断s[0,...,i]是否是回文的方法时间复杂度为$O(n^2)$, 会超时

下面的方法采用KMP算法中最长前缀后缀数组next的概念，时间复杂度为$O(N)$。首先，用一个不在字符串中出现的字符将s和其逆序连在一起,然后求得其next数组，通过next数组的最后一个元素即可找到s中以第一个字符开头的最长回文，然后将剩余部分拼接在其前面。
