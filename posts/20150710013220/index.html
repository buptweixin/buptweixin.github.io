C++ 中的四种类型转换


static_cast 静态类型转换

用语言内建的规则(如int 转为float，char*转为string等)或程序员定义的操作(如类中单参数的构造函数实现的隐式转换)转换一个对象：

float x;
...
cout << static_cast<int>(x); // print x as int
...
f(static_cast<int>("hello")); //使用string而不是char*调用f()

class Stack {
  explicit Stack(int Size);
};
static_cast<Stack>(32); //强制类型转换

dynamic_cast 动态类型转换

将多态类别向下转换为其实际的静态类别（父类转换为具体子类），在四种转换中，它是唯一在执行期间进行检查的转换动作。

class Class;
class Cabriolet: public Car {
  ...
};
class Limousine: public Car {
  ...
};

reinterpret_cast
