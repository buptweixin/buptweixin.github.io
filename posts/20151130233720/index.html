LeetCode 128 Longest Consecutive Sequence

Given an unsorted array of integers, find the length of the longest consecutive elements sequence.

For example,
Given [100, 4, 200, 1, 3, 2],
The longest consecutive elements sequence is [1, 2, 3, 4]. Return its length: 4.

Your algorithm should run in O(n) complexity.

解题思路

  1. 采用unordered_set(set使用红黑树实现，它的find时间复杂度为$O(logN)$)

    1. 将数字存入set关联容器nums_set($O(N)$)
    2. 遍历set中的元素(set的find复杂度$O(1)$)
      1. 如果num-1不在set中，则开始对连续出现数字计数
        1. 初始化int m=num+1
        2. 在nums_set中寻找m，若找到，m+1
        3. 如果没有找到则跳出循环
        4. 更新最长序列为max(longest, m - num)
    class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        set<int> nums_set;
        // 将vector元素复制到set中，方便查找
        nums_set.insert(nums.begin(), nums.end());
        int longest = 0;
        for (int num : nums_set)
        {
            if (nums_set.find(num-1) == nums_set.end())
            {
                int m = num + 1;
                while (nums_set.find(m) != nums_set.end())
                    ++m;
                longest = max(longest, m - num);
            }
        }
        return longest;
    }
};
